# AI Code Review Bundler
**Export your entire codebase as a TXT file for AI code review.**

## ✨ Features

- **Drag & drop files or folders** — recursive scanning
- **File type presets** — Python, JS/TS, Web, JVM, C/C++, Go, Rust, C#, Config, Markdown, TXT
- **Auto-exclude folders** — edit term (default "data"), auto-remove on drop or manual
- **Live tree view** — remove individual files/folders, see stats (included/excluded/size)
- **Export options** — line numbers, table of contents, folder tree


## 🚀 Quick Start

1. Open `code_review_bundler.html` in your browser
1. Drag folders/files or click "browse files"
1. Toggle presets to filter
1. Edit exclude term if needed
1. Click **Download Combined TXT**
1. Copy-paste or attach the file to your conversation with an AI chatbot for code review
